# Vue 3 course. Matheus Battisti - Hora de Codar
Course to learn Vue 3, incorporating modern practices such as 'script setup', TypeScript, SCSS, and more.

```bash
Using {
  Website:
    - Vue 3
    - TypeScript (interfaces + tsx)
    - SCSS
    - Script Setup
    - Vue Router

  Api and DB:
    - JSON Server
}
```

## How to start the application

### Run front-end
```bash
pnpm dev

```

### Run back-end
```bash
pnpm run backend

```
